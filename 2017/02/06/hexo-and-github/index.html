hexo_and_github

2017-02-06
etcblog
blog, hexo

hexo는 jekyll, hugo와 같은 정적 사이트 생성기이다. 간단한 설정만으로 예쁜 사이트를 생성해주고, markdown 문법을 이용해 깔끔한 포스트를 작성할 수 있다. 거기에다가 github pages를 이용하면 무료로 블로그를 운영할 수 있다.

단, 블로그 운영이 WYSIWYG형태로 되어있지 않아, 설정 및 글쓰기가 불편하다는 점과, github에 조금 익숙해야 한다는 점이 사용자에 따라서 단점이 될 수도 있다.
